# 一、环境搭建篇 #

**1.安装node.js**
	
	官网下载安装
	https://nodejs.org/en/


**2.安装less**

	首先通过 
	less
	查看是否已经安装了
	
	npm install -g less
	
**3.安装bowser**

	首先通过 
	bower
	查看是否已经安装了

	npm install -g bower
	
**4.安装grunt**
	
	npm install -g grunt-cli


**用bower下载依赖项**
 	
	cd fork/angulr-2.0.1
	bower
